# R. Courtney Painting — Website

Production website for R. Courtney Painting (San Mateo, CA). Owner: Ryan Courtney. CSLB #918114, serving the Peninsula for 20+ years.

## Stack

- Static HTML/CSS/JS (mirrored from a Webflow design, scrubbed of all template branding)
- Hosted on Netlify
- Forms via Netlify Forms → outgoing webhook → GHL

## Local development

```bash
npx serve .
```

## Re-run the build pipeline

The site was generated by three scripts in `tools/`:

```bash
node tools/mirror.mjs                                      # 1. Mirror source Webflow site
node tools/rebrand.mjs brands/r-courtney-painting.json     # 2. Apply brand find/replace
node tools/finalize.mjs brands/r-courtney-painting.json    # 3. Inject GTM, JSON-LD, meta, wire forms
```

After step 3 also run a one-liner to strip Webflow's SRI integrity hashes (the rebrand modifies CSS and breaks the hashes, so browsers refuse to apply them):

```bash
find . -name "*.html" -exec perl -i -pe 's/\s+integrity="[^"]*"//g; s/\s+crossorigin="anonymous"//g' {} \;
```

## Reuse for another client

Drop a new `brands/<client-slug>.json` next to `r-courtney-painting.json`. Run steps 2 + 3 in a fresh clone.

## Forms

Contact forms (3 across the site) all post to Netlify Forms with `name="contact"`. Submissions consolidate into one inbox.

After deploy:
1. Netlify dashboard → Site → Forms → contact
2. Notifications → Email notification → ryan@ryancourtneypainting.com
3. Optional: Outgoing webhook → GHL inbound webhook URL
